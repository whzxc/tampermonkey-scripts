# 更新日志 (Changelog)

## [1.4.0] - 2026-02-10
### 新增 / Added
- **Emby 剧集详解**: 在详情模态框中显示每一季的详细集数 (e.g., "S1: 8集")。
- **强力计数模式**: 针对 Emby 接口返回集数为 0 的情况，增加手动遍历所有集数进行聚合计算的 fallback 机制。
- **日志增强**: "Process Log" 中的 API 请求现在包含可点击的 URL，且 Emby 响应支持完整对象查看，方便调试。
- **磁力链接优化**: Nullbr 资源列表中的磁力链接增加 "Open" 按钮，方便直接调用本地应用。

### 修复 / Fixed
- **Douban 日志**: 修复了豆瓣详情页和列表页模态框中 "Process Log" 不显示的问题。
- **Emby 计数**: 修复了部分 Emby 库 (如 .strm) 返回集数为 0 的问题。

## [1.3.0] - 2026-02-09
### 新增 / Added
- **DMHY 列表页支持**: 添加 DMHY 列表页 Emby 检测。
- **状态圆点 (Dot)**: 在豆瓣、GYG 和 DMHY 的海报/列表上显示标准化的 Emby 状态圆点。
- **详情模态框 (Modal)**: 点击状态圆点可打开统一的详情模态框，包含 Emby 播放链接、搜索链接 (GYG, BT4G) 和执行日志。

### 变更 / Changed
- 移除了豆瓣详情页的侧边栏面板，替换为海报上的状态圆点。

## [1.2.0] - 2026-02-09
### 新增 / Added
- 使用 Vite 对项目结构进行模块化重构。
- 统建构建流程。
- 将代码拆分为 Services (服务), Handlers (处理器), 和 Utils (工具) 模块。
